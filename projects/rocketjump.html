Rocket Jump

A simple project trying out Unreal Engine 5. I took the first
person template and made a simple game with 2 levels
trying to reach the top of the map as quickly as possible.

The game features 2 levels, with the second being an
identical clone of the first, but featuring a turret
that will fire at the player to make it more challenging.

Each level features a timer to track how quickly you
beat the level which you acomplish by reaching the blue
cube at the very top.

The main menu for the game features an ariel view of the
first level with the moving platforms still working.

All of the controls for the game can be rebinded with the
exception of enter to progress levels and the escape key to
pause.

The turret on level two will fire the same projectiles
as your gun and will knock you off platforms so you have
to be quick.

The game was coded entirely using the Blueprints Visual
Scripting available in Unreal Engine 5. The explosion
and other sound effects were imported with all other
assets being featured in the original demo build.
