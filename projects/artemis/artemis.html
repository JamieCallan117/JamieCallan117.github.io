Artemis

Artemis is a rewrite of the Minecraft mod Wynntils for modern versions of the game. I have made a few contributions to the mod. Some are ports from the original mod, some are new features.

The mod had been released in an Alpha state with a lot of the basics from the original mod, known as legacy, implemented. However, there was a lot from legacy that was yet to be ported over. Over the next few months, features were ported over from various different contributors.

One feature that was missing that I missed having was the waypoint manager, which allowed easy access for viewing and editing your custom waypoints. Currently, the only way to view and edit them was to navigate the map and locate them, then use different input methods to edit or delete them.

The pull request for the feature is available here.

The second feature I worked on is another port from the original mod. The server uses a Minecraft container screen to display various items that can be purchased, in this case passes to travel over the ocean to different islands.

This feature replaced that screen with an overhead map of the world that you can use the mouse to pan the view and see all the possible locations you can travel to. It also adds extra features such as being able to see inaccessible locations and drawing a line between locations.

The feature can be toggled on and off via the in-game config.

The pull request for the feature is available here.

The third feature is essentially the same as the previous, just as an alternative screen. Instead of clicking the area on the map you wanted to travel to, you instead select the destination via a departure board located to the left of the map.

The in-game config allows users to choose whether they want this screen or the basic map screen. Or disable the custom screen entirely.

The pull request for the feature is available here.

The fourth feature I added was another port from the original mod, this time adding a filled arc around an in-game item known as an Emerald Pouch.

An Emerald Pouch allows players to store a large amount of emeralds on them at once. Different tiers allow players to store different amounts, with higher tiers giving more capacity.

Like previous features, the in-game config allows users to choose whether this feature is enabled or not and they can also choose whether the arc will only be rendered on their hotbar or in their inventory.

This feature adds an arc around the item to show how full it is, a small green arc means there is still a lot of space left, a red almost complete arc means it is nearly full.

The pull request for the feature is available here.

This is the first open source project I have contributed to so it has definitely helped me to have a better understanding on working on a bigger project and the process to go through getting a feature merged.

There are still a few more features that need porting from the original mod I may work on, but I also have some ideas for new features that I may try to implement.
